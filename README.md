# Лабораторная работа №12  
**Тема:** Объектно-ориентированное программирование в Python  

## Цель работы
Закрепить навыки работы с классами, наследованием, инкапсуляцией и полиморфизмом. Научиться проектировать систему взаимодействия клиентов и транспортных средств.

---

## Выполненные задания

### Задание 1
- Реализован класс `Client` для клиентов компании.  
- Атрибуты: имя клиента, вес груза, VIP‑статус.  
- Добавлен метод для вывода информации о клиенте.  
- Закрепил навыки создания простых классов и работы с атрибутами.

### Задание 2
- Реализован базовый класс `Vehicle` для транспортного средства.  
- Атрибуты: уникальный идентификатор, грузоподъёмность, текущая загрузка, список клиентов.  
- Метод `load_cargo(client)` проверяет тип данных и не допускает перегрузки.  
- Добавлен магический метод `__str__` для удобного вывода.  
- Закрепил навыки работы с методами, проверкой данных и магическими методами.

### Задание 3
- Созданы классы `Ship` и `Truck`, наследующие от `Vehicle`.  
- У `Ship` добавлен атрибут `name`, у `Truck` — атрибут `color`.  
- Закрепил навыки наследования и переопределения методов.

### Задание 4
- Реализован класс `TransportCompany`.  
- Атрибуты: название компании, список клиентов, список транспортных средств.  
- Методы: добавление клиентов и транспорта, вывод списка, оптимизация распределения грузов.  
- Оптимизация учитывает VIP‑клиентов и минимизирует количество используемого транспорта.  
- В файле `main.py` разработано консольное меню для удобной работы с системой.  
- Закрепил навыки взаимодействия объектов разных классов и построения меню.
